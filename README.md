# KillSleep 
 Wake up when you need to.
 A simple native android app in Java to practice using some android components.
 
 ## Features:
 - Alarm functionality.
 - No snooze button.
 - No alarm dismiss button.
 - To dismiss alarm, touch your phone with a NFC tag far from your bed.

## Relevant topics:
 - AlarmManager.
 - NFC reading.
